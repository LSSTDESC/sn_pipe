Set of scripts to generate LSST-like OS for DDFs
cookbook:
- python run_scripts/desc_ddf_strategy/ddf_cohesive_strategy.py
  -> to generate observing strategies (general) parameters+produces a set of plots
  -> should produce (at least): scenarios_bud.csv where bud is the DD budget

- python run_scripts/desc_ddf_strategy/build_scenarios_from_csv.py --configFile=scenarios_bud.csv --configScenario=input/DESC_cohesive_strategy/config_scenarios.csv
  -> uses scenarios_bud.csv as input
  -> uses configScenario (definition of the field/type/cad/sl)
  -> output (by default in input/DESC_cohesive_strategy): for each scenario: cadence parameters (Nvisits/obs. night, cadence, ...)for each field

- python run_scripts/fakes/make_scenario.py --configFile input/DESC_cohesive_strategy/DDF_DESC_0.80_SN_bud.csv 
  -> will produce LSST-like OS with LSST pointings parameters
  -> this file can be used as "usual" OS to generate and fit LCs
OR
- python run_scripts/fakes/loop_scenarios.py --budget_DD=bud
  -> to loop on possible scenarios



******************************* OR ***************************
to make all the previous stages at once:

- python run_scripts/desc_ddf_strategy/strategy_to_db.py
 -> see --help for possible options.
